# 贝塞尔曲线交互式可视化实验
基于 Python + Taichi GPU 并行框架实现贝塞尔曲线实时绘制，采用 De Casteljau 算法求解曲线坐标，结合底层光栅化、批量显存传输、固定显存对象池优化渲染性能。

## 数学与图形学原理
1. De Casteljau 插值算法给定控制点序列 \(P_0,P_1,...,P_{n-1}\)，参数 \(t \in [0,1]\)，两点线性插值公式：
\(P(t)=(1-t)P_i + tP_{i+1}\)
逐层对相邻控制点插值，每次迭代点数量减一，递归至仅剩单个点，该点即为 t 对应的贝塞尔曲线坐标。
2. 光栅化基础
开辟 800×800 RGBA 帧缓冲区模拟屏幕显存；
算法输出归一化浮点坐标 \([0,1]\)，映射为屏幕整数像素索引；
边界越界判断后修改缓冲区像素颜色，完成像素点亮。
3. 渲染性能优化方案
批量数据传输：CPU 一次性计算全部曲线采样点，单次拷贝至 GPU，降低跨设备通信开销；
GPU 并行光栅化：像素着色逻辑使用 ti.kernel 在 GPU 多线程并行执行；
固定显存池：预分配最大容量缓冲区存储控制点，避免动态内存分配带来的性能损耗。

## 交互操作说明
鼠标左键点击画布：新增红色控制点，自动生成灰色控制多边形辅助线；
实时曲线渲染：控制点数量≥2 时自动绘制绿色贝塞尔曲线，控制点变化实时更新曲线；
键盘 C 按键：清空所有控制点，画布重置为空白。
